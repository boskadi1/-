# v1.39 起始夹爪版本

本工程是从当前 `D:\xiaocheceshi\stm` 独立复制生成的版本，没有覆盖原工程。
夹爪控制只参考用户提供压缩包中的 `grab_servo.c/.h`、舵机通道和角度参数；
参考工程的电机、编码器、循迹等改动均未合入。

## 上电动作

1. TIM1 CH1（PE9，夹爪）和 CH2（PE11，升降）先输出 1500 us 中位。
2. 等待 200 ms，让舵机供电稳定。
3. 夹爪移动到 -38°，升降移动到 0°。
4. 等待 500 ms 到位。
5. 自动闭合夹爪到 30°。
6. 等待 500 ms 后，将升降抬到 15°。
7. 再等待 500 ms，随后允许底盘开始执行任务。

夹爪动作期间底盘始终强制停止。

## 参数

参数集中在 `Core/Inc/car_config.h`：

- `GRAB_STARTUP_AUTO_PICKUP_ENABLE`：上电自动抓取开关。
- `GRAB_CLAW_INIT_DEG`：上电初始张开位置。
- `GRAB_CLAW_CLOSE_DEG`：夹紧位置。
- `GRAB_LIFT_INIT_DEG`：升降初始位置。
- `GRAB_LIFT_UP_DEG`：抓取后的抬升位置。
- `GRAB_ACTION_DELAY_MS`：闭合、抬升各阶段等待时间。

参考代码把闭合目标写为 160°，但同时把最大允许角度设为 30°，实际运行仍会被
限制到 30°。本版本直接使用真实生效值 30°，避免参数含义不一致。

## 蓝牙调试

- `z`：夹爪角度 +5°
- `x`：夹爪角度 -5°
- `u`：升降角度 +5°
- `m`：升降角度 -5°
- `p` 或 `GRAB_PICKUP`：执行闭合、抬升
- `d` 或 `GRAB_DROPOFF`：执行下降、张开

如果实际机械方向相反，应先关闭自动抓取，将角度调整到安全范围后再修改参数。
