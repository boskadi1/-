# v1.56 高频轮速反馈

本版本在 v1.55 基础上提高轮速反馈频率，旧版本保持不变。

## 频率调整

```text
编码器采样及电机速度PID：20 Hz -> 50 Hz
蓝牙轮速回传：          2 Hz  -> 10 Hz
```

对应参数：

```c
#define ENCODER_SAMPLE_MS               20U
#define CAR_BLUETOOTH_TELEMETRY_MS     100U
```

## 非阻塞蓝牙发送

蓝牙仍使用9600波特率。直接每100 ms调用阻塞发送会占用大量主循环时间，因此v1.56改为：

- USART2中断发送；
- 4条消息发送队列；
- 主循环只把遥测数据放入队列，不等待整条数据发送完；
- 电机50 Hz闭环和循迹状态机不会被蓝牙发送长时间阻塞。

蓝牙数据格式不变：

```text
TEL,E=偏差,C=循迹修正,L=目标/反馈/PWM,R=目标/反馈/PWM,N=左右原始脉冲,V=有效标志
```

## 编码器重新标定

采样周期从50 ms改成20 ms，因此满速脉冲默认值按比例从100改为40：

```c
#define ENCODER_COUNTS_PER_SAMPLE_AT_FULL_SPEED 40U
```

实际值必须通过架空测试确定。PWM接近100时读取蓝牙中的`N=左/右`，将稳定的20 ms脉冲数填入该参数。
